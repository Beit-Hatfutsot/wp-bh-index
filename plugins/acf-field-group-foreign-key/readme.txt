Extends ACF plugin with a new field type to provide a linkage to a field group by its foreign custom field

== Description ==

This plugin extends ACF plugin with a new field type to provide a linkage to a field group by its foreign custom field

= Compatibility =

This ACF field type is compatible with:
* ACF 5
* ACF 4

== Installation ==

1. Copy the `acf-field-group-foreign-key` folder into your `wp-content/plugins` folder
2. Activate the Field Group Foreign Key plugin via the plugins admin page
3. Create a new field via ACF and select the Field Group Foreign Key type
4. Read the description above for usage instructions
